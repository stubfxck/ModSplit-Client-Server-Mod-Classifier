# ModSplit-Client-Server-Mod-Classifier
ModSplit (Mod Side Classifier) — инструмент для авторов Minecraft сборок, который определяет, какие моды нужны на клиенте, какие — только на сервере, а какие игрокам ставить не обязательно. Заебался вручную лазить по страницам модов на Modrinth/CurseForge ради этого? Скрипт делает это одной кнопкой.
